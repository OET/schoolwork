NOTE: Some of the interfaces were not implemented due to time constraints

GUI – Custom graphical user interface for each type of user in a restaurant business.
--------------------------------------------------------------
Application Server Interface – The interface handling the HTTPS requests, and authenticating the users, and launching the user’s custom GUI.
--------------------------------------------------------------
Database Interface– All information is stored in a PostgreSQL Database.
---------------------------------------------------------------------------------------------------------------------------

Admin Interface – Enables the system administrator to manage and maintain the system and the user accounts.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Owner Interface – It has master function it can impersonate any type of account in the system and act as that.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Manager Interface – It monitors the entire system. Manages user account and creates reports for the business.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Cashier Interface – The functions of the waiter plus processing the money payments.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Waiter Interface – Manages customer orders and prints out the table bills.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Kitchen Staff Interface – Monitors the orders and notifies when the order is prepared to be taken on the floor.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Employee Accounts Management – Administrating user access levels and user accounts.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

System Management and Maintenance – Administrating the system and managing it. Applying updates, manage features, and view system errors.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Analysis and Reports – Owner or manager can create reports in Word, Excel, or PDF form as well as charts and tables.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Money Transactions –Manager, cashier, and waiter can display customer payment options, manage payout deliveries, and manage general orders.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Order status – Waiter, kitchen staff, and manager can view pending order in system. A unique ID is generated when waiter takes customer order.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Customer order management – Waiter can login, view table, select table
